Resumen del Proyecto: JuvaConnect

JuvaConnect es una plataforma web (Bolsa de Empleo Universitaria) que conecta a estudiantes universitarios en Nicaragua con empresas para oportunidades de pasantías y empleos junior.

1. Arquitectura y Tecnologías

  • Frontend (Cliente): Single Page Application (SPA) construida con HTML5, CSS3 y Vanilla JavaScript (sin frameworks externos complejos).
  • Backend (Servidor): API REST desarrollada en Node.js utilizando el framework Express.
  • Base de Datos: PostgreSQL para el almacenamiento relacional.

2. Estructura de Archivos

  • bolsa_empleo_universitaria.html: Archivo principal de la interfaz. Funciona como SPA, alternando vistas (Landing Page, Panel de Estudiante, Panel de Empresa).
  • styles.css: Reglas de diseño, colores y maquetación visual.
  • script.js: Lógica del navegador (consumo de la API, interacciones y navegación).
  • server.js: Servidor backend que maneja la conexión a la base de datos y provee las rutas de la API.
  • schema.sql: Script de base de datos para la creación de tablas (usuarios, empresas, vacantes, etc.) y generación de datos de prueba.
  • package.json: Configuración del proyecto Node.js y sus dependencias.

3. Funcionalidades Principales

  • Estudiantes: Registro, creación de un CV Digital, búsqueda de ofertas laborales y postulación a vacantes.
  • Empresas: Registro de nuevas vacantes y gestión de los candidatos (aceptar, rechazar o revisar postulaciones).
  • Backend (API REST): Rutas para obtener vacantes (GET /api/jobs), iniciar sesión (POST /api/login), registrar usuarios (POST /api/register) y enviar postulaciones (POST /api/applications).
